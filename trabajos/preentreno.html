Entrenamiento de modelo de lenguaje con texto del Quijote para aprender su estilo literario único

¿Qué es el Pre-entrenamiento?

El pre-entrenamiento consiste en entrenar un modelo con datos específicos para que aprenda un estilo o conocimiento particular. En esta práctica se entrena un modelo "nano" con el texto completo del Quijote de la Mancha para que genere texto con su estilo literario característico.[file:12]

Paso 1: Crear Proyecto

1
mkdir quijote_ia
cd quijote_ia

Crear carpeta quijote_ia e entrar en ella. Aquí se ubicará todo el proyecto de entrenamiento.

Paso 2: Entorno Virtual

2
python -m venv .venv
.venv\Scripts\activate

Crear y activar entorno virtual Python. Aísla las dependencias del proyecto del sistema global.

Paso 3: Instalar PyTorch

3
pip install torch
(.venv) C:\quijote_ia>pip install torch

Instalación exitosa de PyTorch 2.10.0 con todas sus dependencias. Framework esencial para entrenamiento neuronal.[file:12]
