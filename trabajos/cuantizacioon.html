Foto 1: Instalación Ollama en macOS

Primer Paso - Ollama macOS
Captura mostrando el proceso inicial de instalación de Ollama en macOS. Ollama es una plataforma que permite ejecutar modelos de lenguaje grandes (LLMs) localmente de forma sencilla. Esta herramienta es clave para trabajar con modelos cuantizados sin necesidad de servicios en la nube.[file:3]

Foto 2: Comandos Terminal Ollama

Verificación Versión
aulaateca26@Minideaaateca26 ~ % ollama --version
ollama version is 0.14.1

Primera verificación exitosa de Ollama versión 0.14.1. El comando ollama --version confirma que la instalación funcionó correctamente y está lista para descargar y ejecutar modelos.[file:3]

Foto 3: Descarga Modelos IA

Descargando IAs
Interfaz de Ollama mostrando el proceso de descarga de diferentes modelos de IA. Estos modelos se almacenan localmente para ejecución offline. La cuantización permite descargar versiones más ligeras de estos modelos manteniendo gran parte de la calidad.[file:3]

Foto 4: Chat Interactivo Llama3

Prueba Llama3
>>> Hola, ¿quién eres?
Respuesta: "Hola! Soy un modelo de lenguaje entrenado por una inteligencia artificial..."

Prueba exitosa ejecutando ollama run llama3. El modelo responde coherentemente en español, demostrando que Ollama funciona correctamente con modelos grandes sin cuantización.[file:3]

Foto 5: Configuración del Archivo

Archivo de Configuración
Captura del archivo de configuración donde se han definido parámetros específicos del proyecto. Aquí se establecen las rutas, modelos y configuraciones para la comparación de rendimiento con/sin cuantización.[file:3]

Foto 6: Comparación Modelos

Ambos Modelos
Vista comparativa de los dos modelos analizados: uno sin cuantización (tamaño completo, alta precisión) vs uno con cuantización Q4_K_M (tamaño reducido ~4x). Esta es la base del análisis de rendimiento.[file:3]

Comparativa Sin/Con Cuantización

IA Sin Cuantización
Usos: Juegos AAA, edición 4K/8K, renderizado profesional
RTX 4090/3080 Ti: Máximo rendimiento gráficos
RTX 3090: Excelente renderizado avanzado
RTX 7900 XTX: Equilibrio precio/rendimiento

"Mayor tasa refresco (144Hz+), GDDR6, drivers actualizados"[file:3]
IA Con Cuantización
Formato: Q4_K_M.gguf (4 bits)
Ventajas: ~75% menos RAM/VRAM
Velocidad: 2-4x más rápida inferencia
Calidad: 95-98% preservada

Ideal para ordenadores personales y edge computing[file:3]

Foto 7: Análisis GPUs Sin Cuantización

Rendimiento GPUs
Gráfico comparativo de rendimiento de GPUs con -ngl 99 (offload máximo GPU). Muestra FPS, tokens/segundo y uso VRAM para modelos sin cuantización en aplicaciones exigentes.[file:3]

Foto 8: Resultados Con Cuantización

Modelos Cuantizados
Resultados de inferencia con cuantización aplicada. Compara velocidad (t/s), uso memoria y calidad entre modelo original vs modelo Q4_K_M. Demuestra que la cuantización mantiene excelente rendimiento con ~25% memoria.[file:3]

Foto 9: Importación LM Studio

LM Studio GGUF
lms import ~/models/lfm2-1.2b-q4_K_M.gguf --copy

Proceso final: Importar modelo cuantizado a LM Studio para pruebas interactivas. El flag --copy duplica el archivo a la carpeta de modelos automáticamente.[file:3]

Abre LM Studio → "My Models" → busca "lfm2" → Load → ¡Listo para usar!
